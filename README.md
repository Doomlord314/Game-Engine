# Game-Engine
Game Engine For Making Games Quick

# Needed
- SFML
- Code::Blocks
- C++ 14
# Documentation
___

Objects
- x
- y
- width (resx)
- height (resy)
- move Function (move)
- draw (needs sf::RenderWindow)

ObjectManager
- Objects (a C++ Vector)
- Check For Touch Function (isTouch) (needs the indices)
- Add Object Function (push_back)
- A draw function that draws all objects (needs sf::RenderWindow)
